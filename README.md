# Jimini

Jimini is a healthcare-focused project with a backend API and a React Native mobile app.


## Directory Roles

- `api/`: Backend API service source code.
- `api/src/`: API implementation modules (routes, business logic, data access).
- `app/mobile/`: React Native mobile app project.
- `scripts/`: Utility scripts for development/operations.
